# RNK™ Cyphur

RNK™ Cyphur is an encrypted communications module for Foundry VTT with private messaging, group chats, image sharing, export tools, and GM monitoring utilities.

## Features

- Private player-to-player messaging
- Group chat creation and management
- Image sharing inside conversations
- GM monitor and moderation utilities
- Local file and journal export options
- System-agnostic UI for Foundry VTT v13

## Installation

Manifest URL:

`https://github.com/RNK-Enterprise/rnk-cyphur/releases/latest/download/module.json`

1. Open Foundry VTT.
2. Go to **Add-on Modules**.
3. Select **Install Module**.
4. Paste the manifest URL.
5. Complete installation and enable `RNK™ Cyphur` for the world.

## Development

Validation is handled locally with the built-in repo validator.

- `npm run validate`
- `npm run test`
